AU Organisations Register

This is a Linked Data version of the contents of directory.gov.au. It returns very simple web pages (HTML) or machine-readable, Resource Description Framework (RDF), content. Internet protocol content negotiation or Alternates View query strings can be used to request the different formats.

See the About Page for more info.

Content

1. An example organisation persistent URI (web address) for the CSIRO is:

  • http://test.linked.data.gov.au/org/O-000886
    • Machine-readable RDF: http://test.linked.data.gov.au/org/O-000886?_format=text/turtle

...where O-000886 is the Department of Finance's unique record ID for the CSIRO - it's persistent ID.

2. An example of a Register - a list of items - is the Persons Register:

  • http://test.linked.data.gov.au/person/
    • Machine-readable RDF: http://test.linked.data.gov.au/person/?_format=text/turtle

...where the web address is not for a specific Person but a list of persons.
